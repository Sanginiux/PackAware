# PackAware

PackAware is a Python-based application designed to analyze the environmental impact of plastic packaging and encourage sustainable decision-making. The project evaluates different packaging materials, provides insights into their ecological footprint, and recommends environmentally friendly alternatives to reduce plastic waste.

Developed during **Algo Verse 2025**, organized by **HackWithIndia** in collaboration with **IGDTUW**, the project focuses on raising awareness about sustainable packaging through data-driven analysis.

---

## Features

- Analyze different types of plastic packaging
- Evaluate environmental impact based on predefined metrics
- Compare packaging materials
- Recommend eco-friendly alternatives
- User-friendly interface
- Fast and efficient analysis
- Modular and scalable architecture

## How It Works

1. User provides packaging information.
2. The system analyzes the packaging material.
3. Environmental impact metrics are calculated.
4. Sustainable alternatives are identified.
5. Results are displayed with recommendations.

## Installation

### Clone the repository

```bash
git clone https://github.com/Sanginiux/PackAware.git
```

### Navigate to the project

```bash
cd PackAware
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python main.py
```
